Arch Linux is an independently developed, x86-64 general-purpose GNU/Linux
distribution that strives to provide the latest stable versions of most software
by following a rolling-release model.

The mirrors listed below host the official package repositories. When
configuring your `/etc/pacman.d/mirrorlist`, choosing an "Online" server close
to your location will ensure fast and reliable system updates.

See the [ArchLinux wiki](https://wiki.archlinux.org/title/Mirrors) for official
and up-to-date information.
